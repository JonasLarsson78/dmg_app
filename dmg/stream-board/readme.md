# Stream Board — DMG Distribution

---

## Svenska

Installationsfil (disk image) för Stream Board-appen till macOS.

### Fil

`Stream Board_0.1.0_aarch64.dmg` — version 0.1.0, Apple Silicon (aarch64)

### Installation

1. Öppna `.dmg`-filen
2. Dra **Stream Board.app** till Applications-mappen
3. Mata ut disk imagen
4. Starta Stream Board från Applications eller Spotlight

### Krav

- macOS 12.0 eller senare
- Apple Silicon (M1 eller senare)

---

## English

Installer disk image for the Stream Board macOS app.

### File

`Stream Board_0.1.0_aarch64.dmg` — version 0.1.0, Apple Silicon (aarch64)

### Installation

1. Open the `.dmg` file
2. Drag **Stream Board.app** to the Applications folder
3. Eject the disk image
4. Launch Stream Board from Applications or Spotlight

### Requirements

- macOS 12.0 or later
- Apple Silicon (M1 or later)

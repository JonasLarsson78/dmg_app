# Port Manager

---

## Svenska

En macOS-app för att hantera lokala utvecklingsportar. Byggd med Tauri.

### Krav

- macOS 12.0 eller senare
- Apple Silicon (arm64)

### Installation

Kopiera `Port Manager.app` till din `/Applications`-mapp.

### Användning

Starta Port Manager från Applications eller Spotlight.

- Listar alla aktiva portar på din dator med processinformation
- Avsluta valfri process direkt i appen med ett klick

### Detaljer

- **Version**: 0.1.0
- **Bundle ID**: dev.portmanager.app
- **Arkitektur**: aarch64 (Apple Silicon)
- **Ramverk**: Tauri

---

## English

A macOS desktop app for managing local development ports. Built with Tauri.

### Requirements

- macOS 12.0 or later
- Apple Silicon (arm64)

### Installation

Copy `Port Manager.app` to your `/Applications` folder.

### Usage

Launch Port Manager from Applications or Spotlight.

- Lists all active ports on your machine with process info
- Kill any process directly from the app with one click

### Details

- **Version**: 0.1.0
- **Bundle ID**: dev.portmanager.app
- **Architecture**: aarch64 (Apple Silicon)
- **Framework**: Tauri
